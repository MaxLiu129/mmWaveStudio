=========== MMWAVE STUDIO Release 02.01.01 ===================

TI mmWave Studio enables characterization and evaluation of 
TI mmWave sensors. 
It captures raw ADC data from xWR1XXX EVM using
1. DCA1000 capture card or TSW1400 for single chip devices.
2. TDA2XX capture card for cascade chip devices. 
It includes necessary components to process the data and 
visualize the processed data.
==============================================================

New features in this release:

- Updated FTDI drivers (Signed with Validity till November 2021)
- Update RF Evaluation image for 6843 device

Refer to Release notes for list of Bug-Fixes and known issues
